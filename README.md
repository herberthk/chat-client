## Private Realtime chat application

> Live chat is the real time chat web application that supports private chat rooms developed using websockets

### Tech-stack front-end

- Reactj.js
- Socket.io
- Custom css
- Github actions for CI/CD

### Backend

- Typescript
- Node.js/express
- Socket.io
- Jest for testing
